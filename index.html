Controls

    Player 1

  • Move Right : D
  • Move Left : A
  • Jump : W
  • Block : S
  • Punch/Select Character: C
  • Kick/Deselect Character: V
  • Special Attack : B

    Player 2

  • Move Right : Right Arrow
  • Move Left : Left Arrow
  • Jump : Up Arrow
  • Block : Down Arrow
  • Punch/Select Character: J
  • Kick/Deselect Character: K
  • Special Attack : L

    Misc

  • Pause / Play Music: M
